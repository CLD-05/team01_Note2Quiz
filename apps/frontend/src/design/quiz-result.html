객관식

Q1. 다음 중 Docker 컨테이너의 특징으로 올바른 것은?

해설: Docker 컨테이너는 호스트 OS의 커널을 공유하여 가상 머신보다 가볍고 빠르게 실행됩니다.

객관식

Q2. Docker 컨테이너와 가상 머신의 차이점으로 올바른 것은?

해설: Docker 컨테이너는 하이퍼바이저 없이 호스트 OS의 커널을 공유하므로 가상 머신보다 훨씬 가볍고 시작 속도가 빠릅니다.

객관식

Q3. Docker 이미지와 컨테이너의 관계로 올바른 것은?

해설: Docker 컨테이너는 이미지를 기반으로 생성된 격리된 실행 환경이며, 하나의 이미지로 여러 개의 컨테이너를 생성할 수 있습니다.
